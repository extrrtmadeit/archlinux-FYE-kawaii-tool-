- `extort_fye_menu_v3.sh` - advanced FYE menu version
- `hackmenu.sh` - legal security helper menu
- `kawaii_pack.sh` - kawaii commands/aliases pack
- `kawaii_pack_patch.sh` - patch to disable fish spam in kawaii mode
- `extort_banner.sh` - EXTORT startup banner script
- `drip_pack_v2.sh` - extra vibe aliases and scripts
- `extort_offline_hacker.py` - fullscreen hacker-style screen
- `masterl_cyber_glass_metal.html` - cyber glass metal animation page

Use these as references or copy pieces into `setup.sh` if you want one unified installer.
