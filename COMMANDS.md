# FYE Commands

## Core
- `fye` - Main interactive menu
- `hackmenu` - Legal security helper menu

## Style / Fun
- `banner` - Big EXTORT banner
- `matrix` - Matrix rain animation
- `nyan` - Nyan cat animation
- `kawaii` - Kawaii scene
- `pipes` - Color pipes animation
- `wisdom` - Random quote in color

## Useful
- `hacktop` - btop monitor
- `repo` - onefetch summary in git repo
- `ranger` - file manager
- `dust` - disk usage by folder
- `duf` - filesystem usage
- `jq` - JSON formatter

## Quick start
```bash
chmod +x setup.sh
./setup.sh
source ~/.zshrc
fye
```
