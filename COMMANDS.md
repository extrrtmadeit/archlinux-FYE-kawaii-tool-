# FYE Commands

## Core
- `fye` - Main interactive menu (Ultimate build supports 30+ options)
- `hackmenu` - Legal security helper menu

## Style / Fun
- `banner` - Big EXTORT banner
- `matrix` - Matrix rain animation
- `nyan` - Nyan cat animation
- `kawaii` - Kawaii scene
- `pipes` - Color pipes animation
- `wisdom` - Random quote in color

## Useful
- `hacktop` - btop monitor
- `repo` - onefetch summary in git repo
- `ranger` - file manager
- `dust` - disk usage by folder
- `duf` - filesystem usage
- `jq` - JSON formatter

## Integrations (inside fye)
- set/send Discord webhook messages
- set/send Telegram bot messages
- open Discord/Telegram web

## SFX
Inside fye menu:
- Generate local SFX
- Optional online SFX fetch

## Quick start
```bash
chmod +x setup-v2.sh
./setup-v2.sh
source ~/.zshrc
fye
```
